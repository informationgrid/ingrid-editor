# Koordinatenreferenzsystem

Angabe des Koordinatenreferenzsystems.

Für INSPIRE-interoperable Datensätze muss mindestens ein Eintrag mit einem EPSG-Code eingetragen sein.

## Beispiel:

EPSG:4326 / WGS 84 / geographisch
