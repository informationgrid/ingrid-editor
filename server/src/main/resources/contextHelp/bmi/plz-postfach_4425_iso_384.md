# PLZ (Postfach)

Postleitzahl des Postfaches. Hier ist die Postleitzahl einzugeben, die sich auf das Postfach bezieht. Für die Postleitzahl zur Hausadresse (Straße/Hausnummer) ist ein eigenes Feld vorgesehen.

## Beispiel:

8977
