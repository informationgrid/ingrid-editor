
## Documents

### GET dataset/find?query=<string>&sort=<field>:<[asc|desc]

* Dashboard
  * get recent documents
 
### GET datasets/<id>

* Form
  * load document by clicking on tree/browser or by URL parameter
  
### POST datasets/<id>?publish=<boolean> -> <body>

* Form: save document

### DELETE datasets/<id>

* Form: delete document
